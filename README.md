# Headphone-design
CAD Project
Wireless Headphones aka Kabellose Kopfhörer

## Team-Members
CADAdv_Team_4: Hazeeq,Isma and Syahir



## English

### About the Project
This project is about designing wireless headphones in CATIA 3DEXPERIENCE.  
The main goal is to create a parametric CAD model that can be changed easily by adjusting important dimensions.

### Project Goals
- Create a parametric model of wireless headphones
- Control important dimensions with parameters
- Make changes and new variants more easily
- Consider manufacturing and assembly during the design process

### Main Requirements
- The headphones should be modeled parametrically
- Dimensions such as the headband radius and earcup width should be adjustable
- A mechanical connection between the earcups and the headband should be included
- The earcups should be able to move in a defined way, for example by rotating
- The movement should be suitable for simulation and analysis
- The earcups should fully cover the ears (over-ear design)
- The design should be suitable for manufacturing and assembly

### Software Used
- CATIA 3DEXPERIENCE

### Result
The result of this project is a structured CAD model of wireless headphones with adjustable parameters and a design that meets the defined requirements.

## Deutsch

### Über das Projekt
In diesem Projekt werden kabellose Kopfhörer in CATIA 3DEXPERIENCE konstruiert.  
Das Hauptziel ist ein parametrisches CAD-Modell, das sich durch die Anpassung wichtiger Maße einfach ändern lässt.

### Projektziele
- Erstellung eines parametrischen Modells für kabellose Kopfhörer
- Steuerung wichtiger Maße über Parameter
- Einfachere Umsetzung von Änderungen und Varianten
- Berücksichtigung von Fertigung und Montage in der Konstruktion

### Wichtige Anforderungen
- Die Kopfhörer sollen parametrisch modelliert werden
- Maße wie Kopfbügelradius und Ohrmuschelbreite sollen anpassbar sein
- Zwischen Ohrmuschel und Kopfbügel soll eine mechanische Verbindung vorhanden sein
- Die Ohrmuscheln sollen eine definierte Beweglichkeit besitzen, zum Beispiel durch Drehen
- Die Bewegung soll für Simulation und Analyse geeignet sein
- Die Ohrmuscheln sollen die Ohren vollständig umschließen (Over-Ear-Design)
- Die Konstruktion soll fertigungs- und montagegerecht sein

### Verwendete Software
- CATIA 3DEXPERIENCE

### Ergebnis
Das Ergebnis dieses Projekts ist ein strukturierter CAD-Entwurf kabelloser Kopfhörer mit anpassbaren Parametern und einer Konstruktion, die die festgelegten Anforderungen erfüllt.
